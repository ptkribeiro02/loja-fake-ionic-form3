# Loja Fake - Ionic Angular

Aplicação desenvolvida em Ionic + Angular consumindo a FakeStore API.

## Objetivo

O projeto foi desenvolvido para a disciplina da faculdade com o objetivo de praticar:

- Consumo de API REST
- Ionic Framework
- Angular Standalone
- Componentes Ionic
- Navegação por Tabs
- Consumo de dados HTTP

---

## API Utilizada

FakeStore API:

https://fakestoreapi.com/products

---

## Funcionalidades

### Tab 1
- Informações do aluno
- Foto pessoal
- Turma, unidade e turno

### Tab 2
- Exibição individual dos produtos
- Navegação entre produtos
- Botões desabilitados no início e final da lista

### Tab 3
- Exibição de todos os produtos
- Botão para mostrar produtos
- Botão para ocultar produtos
